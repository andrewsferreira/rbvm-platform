---

## Como funciona

A engine de i18n está em `src/config/i18n.js` e expõe a função `t(key)`:

```javascript
import { t } from './config/i18n.js';

t('nav.dashboard')           // → "Dashboard"
t('settings.theme')          // → "Tema da Interface" (pt-BR) ou "Interface Theme" (en-US)
t('topbar.critical')         // → "Críticas" ou "Critical"
```

**Fallback**: se uma chave não existe no idioma atual, tenta `pt-BR`, depois retorna a própria chave.

---

## Arquivos de tradução

```
src/i18n/pt-BR.js   ← Português (padrão)
src/i18n/en-US.js   ← English
```

Cada arquivo exporta um objeto `PT_BR` ou `EN_US` com pares chave-valor:

```javascript
export const PT_BR = {
  'nav.dashboard':     'Dashboard',
  'settings.theme':    'Tema da Interface',
  'topbar.critical':   'Críticas',
  // ...
};
```

---

## Como trocar de idioma

**Via UI:** Settings → Idioma → clicar em Português ou English.

**Via código:**
```javascript
window.setLanguage('en-US'); // ou 'pt-BR'
```

**Internamente:**
```javascript
window.setLanguage = lang => {
  setLang(lang);
  savePreference('language', lang);
  render(); // re-renderiza com novo idioma
};
```

---

## Persistência

O idioma escolhido é salvo em `localStorage` com a chave `rbvm_lang` e restaurado em `initApp()`.

---

## Como adicionar uma nova tradução

1. Abrir `src/i18n/pt-BR.js` e adicionar a chave em português:
   ```javascript
   'modulo.minhaChave': 'Meu texto em português',
   ```

2. Abrir `src/i18n/en-US.js` e adicionar a chave em inglês:
   ```javascript
   'modulo.minhaChave': 'My English text',
   ```

3. Usar nos templates JS:
   ```javascript
   `<div class="card-t">${t('modulo.minhaChave')}</div>`
   ```

## Convenções de nomenclatura de chaves

| Prefixo | Uso |
|---|---|
| `nav.*` | Labels de navegação e seções da sidebar |
| `topbar.*` | Labels da barra superior |
| `settings.*` | Textos do módulo de configurações |
| `action.*` | Botões de ação comuns |
| `status.*` | Labels de status |
| `sev.*` | Labels de severidade |
| `footer.*` | Rodapé |
| `disclaimer.*` | Avisos e disclaimers |
| `dashboard.*` | Módulo Dashboard |
| `vulns.*` | Módulo Vulnerabilidades |
| *(módulo).*` | Chaves específicas de cada módulo |
